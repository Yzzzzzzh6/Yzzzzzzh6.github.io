# 回声谷网站 - 完整项目文档

## 项目概述

回声谷是一个专为大学生打造的安全、温暖、深度的校园情感社区网站。基于策划案完整实现，包含五大核心模块：首页内容聚合、心声树洞UGC内容、共鸣广场话题讨论、心灵驿站专业支持和个人中心用户管理。

## 核心功能

### 🏠 首页 - 内容聚合与智能推荐
- **智能推荐流**: 基于情绪识别和兴趣标签推荐内容
- **校园声量热力图**: 可视化展示校园情绪分布
- **AI情感分析仪表盘**: 实时展示平台情感分析数据
- **热门话题榜**: 每周更新的校园热点话题

### 🌳 心声树洞 - UGC内容生产
- **加密投稿系统**: 安全匿名的内容发布
- **AI回声仪**: 基于NLP技术生成温暖回应
- **情绪星系**: 将用户情绪转化为可视化图案
- **内容分类标签**: 多维度内容发现和筛选

### 🗣️ 共鸣广场 - 话题讨论
- **话题熔炉**: 算法聚合相似话题
- **观点碰撞**: 支持正反方观点PK
- **回声认证**: 优质内容创作者认证体系
- **热门话题榜**: 每周更新校园热点话题

### 🏥 心灵驿站 - 专业支持
- **解忧盲盒**: 随机获取心理支持资源
- **导师灯塔**: 认证心理咨询师在线答疑
- **心理资源库**: 专业心理健康资源集合
- **危机干预**: AI识别高危言论自动标记

### 👤 个人中心 - 用户管理
- **成长体系**: 回声值、成就徽章、等级系统
- **隐私设置**: 灵活的隐私控制选项
- **数据统计**: 个人使用数据和情绪分析
- **情绪趋势**: 情绪变化追踪图表

## 使用说明

### 本地运行
1. 将所有文件下载到本地目录
2. 在项目根目录启动HTTP服务器：
   ```bash
   python -m http.server 8000
   ```
3. 在浏览器中访问：`http://localhost:8000`
