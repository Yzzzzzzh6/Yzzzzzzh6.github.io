# 回声谷网站设计文档

## 设计理念

### 核心概念
- **温暖共鸣**：体现"你的声音，必有回响"的核心理念
- **安全信任**：营造安全匿名的倾诉环境
- **现代简约**：清晰的信息架构，减少视觉噪音
- **情感可视化**：将抽象情绪转化为美丽图案

### 色彩方案
- **主色调**：温暖的米白色 (#FAF9F6) 作为背景
- **强调色**：柔和的蓝绿色 (#4A90A4) - 代表平静与治愈
- **辅助色**：温暖的珊瑚色 (#E8A598) - 代表温暖与关怀
- **文字色**：深灰色 (#2C3E50) - 确保可读性
- **卡片背景**：纯白色 (#FFFFFF) - 突出内容

### 字体设计
- **标题字体**：Noto Serif SC - 优雅衬线字体，传达专业与温暖
- **正文字体**：Noto Sans SC - 清晰无衬线字体，确保可读性
- **装饰字体**：ZCOOL KuaiLe - 用于Logo和特殊标题

### 视觉语言
- **圆润设计**：大量使用圆角，营造柔和感
- **卡片布局**：信息模块化，清晰分层
- **渐变效果**：微妙的渐变增加视觉层次
- **动画效果**：柔和的过渡动画，增强交互体验

## 视觉效果

### 使用的库和效果
1. **Anime.js** - 页面元素动画和交互反馈
2. **ECharts.js** - 校园声量热力图和情绪可视化
3. **p5.js** - 创意编程实现情绪星系可视化
4. **Splide.js** - 内容轮播和图片展示
5. **Shader-park** - 背景视觉效果
6. **Matter.js** - 物理动画效果
7. **PIXI.js** - 高性能图形渲染

### 动画效果
- **页面加载**：元素渐入动画
- **滚动动画**：内容区域平滑显现
- **悬停效果**：卡片轻微上浮和阴影变化
- **按钮交互**：颜色渐变和轻微缩放

### Hero区域效果
- **背景**：使用p5.js创建动态的情绪可视化粒子系统
- **标题**：渐变色文字动画
- **描述文字**：打字机效果

### 特色组件
1. **情绪星系** - 使用p5.js将用户情绪转化为动态星系图案
2. **校园声量热力图** - ECharts实现的数据可视化
3. **AI回声仪** - 模拟AI回复的动画效果
4. **声波可视化** - 实时音频可视化效果

### 交互设计
- **投稿系统**：多步骤表单，实时预览
- **话题讨论**：动态加载和实时更新
- **个人中心**：情绪变化趋势图表
- **搜索功能**：智能提示和高亮显示

### 响应式设计
- **桌面端**：多列布局，充分利用屏幕空间
- **平板端**：两列布局，保持内容可读性
- **移动端**：单列布局，优化触摸交互

## 页面结构

### 导航结构
- 首页（内容聚合）
- 心声树洞（UGC内容）
- 共鸣广场（话题讨论）
- 心灵驿站（专业支持）
- 个人中心（用户管理）

### 页面布局
- **顶部导航**：固定导航栏，包含Logo和主要功能入口
- **Hero区域**：品牌展示和核心功能介绍
- **内容区域**：模块化内容展示
- **侧边栏**：辅助功能和推荐内容
- **底部**：版权信息和联系方式
