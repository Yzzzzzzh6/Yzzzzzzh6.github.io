# 回声谷网站交互设计文档

## 核心交互功能

### 1. 首页 - 内容聚合与智能推荐

#### 1.1 智能推荐流
- **功能描述**：基于用户兴趣和行为推荐个性化内容
- **交互设计**：
  - 瀑布流布局，无限滚动加载
  - 每个内容卡片包含：标题、摘要、情绪标签、互动数据
  - 悬停显示完整内容和快速操作按钮
  - 右键菜单：收藏、举报、屏蔽同类
- **数据模拟**：
  - 15+不同类型的内容卡片（情感倾诉、经验分享、心理科普等）
  - 实时更新的互动数据（点赞、评论、共鸣数）

#### 1.2 校园声量热力图
- **功能描述**：可视化展示校园情绪分布和热点话题
- **交互设计**：
  - 使用ECharts.js创建交互式地图
  - 点击学校查看详细情绪数据
  - 时间轴滑块：查看不同时间段的数据变化
  - 话题标签云：点击查看相关讨论
- **数据可视化**：
  - 情绪热力图（焦虑、开心、压力、孤独等）
  - 话题趋势折线图
  - 学校对比柱状图

#### 1.3 AI情感分析仪表盘
- **功能描述**：实时展示平台情感分析数据
- **交互设计**：
  - 动态数据卡片，实时更新
  - 情绪分布饼图
  - 高危内容预警提示
  - 点击查看详细分析报告

### 2. 心声树洞 - UGC内容生产

#### 2.1 加密投稿系统
- **功能描述**：安全匿名的内容发布系统
- **交互设计**：
  - 多步骤表单：选择分类 → 编写内容 → 情绪标记 → 发布设置
  - 实时字数统计和格式预览
  - 情绪标签选择器：支持多选和自定义
  - 隐私设置：完全匿名、仅对校友可见、指定时间后公开
  - 发布前预览和编辑功能
- **内容类型**：
  - 文字投稿（支持Markdown格式）
  - 图片上传（支持多张）
  - 语音倾诉（模拟录音功能）

#### 2.2 AI回声仪
- **功能描述**：AI智能生成温暖回应
- **交互设计**：
  - 内容发布后自动触发AI回复（2-3秒延迟模拟思考）
  - 打字机效果显示AI回复内容
  - 回复包含：情感共鸣、建议指导、相关资源推荐
  - 用户对AI回复的反馈（有帮助/需要人工回复）
  - 触发真人互助机制的提示

#### 2.3 情绪星系
- **功能描述**：将用户情绪转化为可视化星系图案
- **交互设计**：
  - 使用p5.js创建动态星系可视化
  - 不同情绪对应不同的颜色和粒子运动
  - 点击星系查看对应的情感日记
  - 时间轴查看情绪变化轨迹
  - 支持分享情绪星系图片

#### 2.4 内容分类与筛选
- **功能描述**：多维度内容发现和筛选
- **交互设计**：
  - 侧边栏分类标签：学业压力、人际关系、情感困惑、家庭问题、就业焦虑
  - 排序选项：最新、最热、共鸣最多、AI推荐
  - 搜索功能：关键词搜索+智能提示
  - 筛选器：时间范围、学校、情绪类型

### 3. 共鸣广场 - 话题讨论

#### 3.1 话题熔炉
- **功能描述**：算法聚合相似话题，展示同类困惑百分比
- **交互设计**：
  - 话题卡片显示：标题、参与人数、同类困惑百分比
  - 点击进入话题详情页
  - 支持关注话题和接收更新通知
  - 话题热度实时更新

#### 3.2 观点碰撞（PK模式）
- **功能描述**：支持正反方观点辩论
- **交互设计**：
  - 选择立场：支持/反对/中立
  - 发布观点：文字+图片+引用
  - 实时投票和观点统计
  - 优质观点获得"金话筒"标识
  - 生成话题报告和结论总结

#### 3.3 回声认证系统
- **功能描述**：优质内容创作者认证体系
- **交互设计**：
  - 用户等级显示：新声、回响、共鸣、传声
  - 回声值进度条和等级特权说明
  - 成就徽章展示：助人为乐、优质创作者、活跃分子
  - 认证用户内容优先展示标识

#### 3.4 热门话题榜
- **功能描述**：每周更新的校园热点话题榜单
- **交互设计**：
  - 排行榜样式展示TOP 20话题
  - 话题趋势箭头和热度变化
  - 点击查看话题详细数据和讨论
  - 生成《校园情感白皮书》下载链接

### 4. 心灵驿站 - 专业支持

#### 4.1 解忧盲盒
- **功能描述**：随机获取心理支持资源
- **交互设计**：
  - 盲盒抽取动画效果
  - 随机获取：呼吸练习、治愈漫画、音乐推荐、心理测试
  - 每种资源类型有独特的展示界面
  - 收藏喜欢的资源和创建个人资源库
  - 分享资源到社区

#### 4.2 导师灯塔
- **功能描述**：认证心理咨询师和优秀学长学姐在线答疑
- **交互设计**：
  - 导师列表：头像、认证标识、专业领域、评价
  - 在线状态指示和预约系统
  - 匿名问答界面：支持文字和图片提问
  - 问答历史记录和收藏
  - 导师评价和推荐系统

#### 4.3 心理资源库
- **功能描述**：专业心理健康资源集合
- **交互设计**：
  - 分类浏览：心理援助热线、咨询中心信息、科普文章
  - 地图模式显示附近的心理咨询资源
  - 资源收藏和分享功能
  - 紧急求助一键拨打功能

#### 4.4 危机干预系统
- **功能描述**：AI识别高危言论并提供专业支持
- **交互设计**：
  - 高危内容自动标记和隐藏
  - 自动推送专业心理援助资源
  - 紧急联系人设置和一键求助
  - 24小时在线客服支持

### 5. 个人中心 - 用户管理

#### 5.1 成长体系
- **功能描述**：用户等级和成就系统
- **交互设计**：
  - 等级进度条和升级动画
  - 成就徽章收集和展示
  - 回声值获取记录和用途说明
  - 等级特权介绍和解锁提示

#### 5.2 隐私设置
- **功能描述**：灵活的隐私控制选项
- **交互设计**：
  - 动态虚拟头像生成器
  - 匿名/实名切换开关
  - 内容可见性设置矩阵
  - 数据导出和删除功能

#### 5.3 数据统计
- **功能描述**：个人使用数据和情绪分析
- **交互设计**：
  - 发帖数、获赞数、回声值统计卡片
  - 情绪变化趋势图表（ECharts.js）
  - 使用时长统计和活动热力图
  - 个人影响力分析报告

## 通用交互功能

### 搜索与筛选
- 全局搜索：支持关键词、标签、用户搜索
- 智能提示：搜索时显示相关推荐
- 搜索结果排序：相关度、时间、热度

### 通知系统
- 消息中心：点赞、评论、系统通知
- 实时通知：WebSocket模拟实时消息推送
- 通知设置：自定义接收通知类型

### 社交互动
- 点赞/共鸣：点击动画效果
- 评论/回复：支持表情和图片
- 收藏/分享：收藏夹管理和分享卡片生成

### 响应式交互
- 移动端优化：触摸友好的交互设计
- 手势支持：滑动切换、长按操作
- 离线提示：网络断开时的友好提示

## 技术实现要点

### 动画效果
- 使用Anime.js实现页面过渡和元素动画
- p5.js创建创意可视化和粒子效果
- Matter.js添加物理动画效果

### 数据可视化
- ECharts.js创建图表和热力图
- 实时数据更新和动画过渡
- 交互式图表支持缩放和筛选

### 模拟数据
- 创建丰富的模拟数据支持各种交互场景
- 模拟用户行为和数据变化
- 支持数据的增删改查操作
